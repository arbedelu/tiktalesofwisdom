# TikTales: Filipino Proverbs in the Era of Social Media

*TikTales* is a digital catalog and interactive web application that explores how traditional Filipino proverbs (salawikain) have evolved, adapted, and recirculated within modern digital spaces and social media platforms. 

---

## About the Project

Traditional Filipino proverbs serve as generational moral guides, reflecting cultural values and social norms. However, in modern digital contexts, these expressions undergo creative shifts—often mutating into memes, humorous status updates, or romantic warnings while playing on the structure of the original phrases.

This project archives, contrasts, and visualizes the shifting meanings of these cultural anchors from their traditional roots to their digitalized, internet-era counterparts.
